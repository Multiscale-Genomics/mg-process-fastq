MuG - FastQ Pipelines's documentation!
======================================
**Quick Links**: `Source (github) <https://github.com/Multiscale-Genomics/mg-process-fastq>`_ - `Documentation (rtd) <https://mg-process-fastq.readthedocs.io/>`_

Requirements
------------

- bedToBigBed - http://hgdownload.cse.ucsc.edu/admin/exe/linux.x86_64/
- HDF5
- Python 2.7.10+
- numpy
- h5py
- pysam

Installation
------------
Directly from GitHub:

.. code-block:: none
   :linenos:
   
   git clone https://github.com/Multiscale-Genomics/mg-process-fastq.git

Using pip:

.. code-block:: none
   :linenos:

   pip install git+https://github.com/Multiscale-Genomics/mg-process-fastq.git
